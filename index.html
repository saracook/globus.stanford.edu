Globus is a high-performance data-transfer platform. You can use Globus for…

  • Data transfers between servers in your group.
  • Data transfers between a server and your laptop.
  • Sharing data with researchers at other institutions.
  • Sharing data with the world.
  • Automate data transfers between any of the above.

Transfers happen unattended, data verification is on by default, and on-the-wire encryption is available.

All of this is available now, for you, for free.

Get Started
